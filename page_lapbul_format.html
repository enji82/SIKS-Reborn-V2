Format Laporan Bulan

Pusat Template

Gunakan format resmi untuk keselarasan data sistem.

Petunjuk Pengerjaan
  • Unduh file format Excel melalui tombol utama di bawah ini.
  • Isi data laporan bulanan sekolah Anda dengan lengkap sesuai petunjuk kolom di dalam file.
  • Penting: Jangan menambah/menghapus kolom agar data terbaca otomatis oleh OCR sistem.
  • Scan hasil cetak ke format PDF (Maks 1 MB) lalu unggah di menu Kelola Data.
